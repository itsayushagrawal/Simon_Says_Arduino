# Bill of Materials (BOM)

This document lists all the hardware required to build the Arduino Simon Says Game.

---

## Components

| Item | Quantity | Description |
|------|:--------:|-------------|
| Arduino Uno | 1 | Main microcontroller board |
| Breadboard | 1 | Full-size breadboard |
| LED (Any Color) | 4 | Game LEDs |
| Push Button | 4 | User input buttons |
| 220Ω Resistor | 4 | LED current limiting resistors |
| 10kΩ Resistor | 4 *(Optional)* | Only required if not using `INPUT_PULLUP` |
| Passive Buzzer | 1 | Audio feedback |
| 74HC595 Shift Register | 2 | Drives the dual 7-segment display |
| Common Anode 7-Segment Display | 2 | Displays the current score |
| Jumper Wires | As Required | Breadboard connections |
| USB Cable | 1 | Arduino programming and power |

---

## Software

- Arduino IDE
- Wokwi Simulator (Optional)
